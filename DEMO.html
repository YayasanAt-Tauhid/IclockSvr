üì¶ Cara Install Python 3

Langkah 1: Download Python

Download Python 3.11 dari website resmi:

https://www.python.org/downloads/windows/

Atau klik tombol di bawah:

Download Python 3.11.7

Langkah 2: Install Python

Jalankan installer yang sudah didownload:

  • ‚úÖ Centang "Add Python to PATH"
  • Klik "Install Now"
  • Tunggu hingga selesai
  • Restart Command Prompt

Langkah 3: Verifikasi Instalasi

Buka Command Prompt dan cek versi Python:

python --version

Harus menampilkan: Python 3.11.x

Langkah 4: Jalankan Setup

Masuk ke folder aplikasi dan jalankan setup:

cd C:\iclockSvr\linux_version setup-windows.bat

Langkah 5: Buat Admin User

Ikuti prompt untuk membuat superuser:

python manage.py createsuperuser

Masukkan username, email, dan password

Langkah 6: Jalankan Server

Start development server:

python manage.py runserver

Buka browser: http://127.0.0.1:8000
